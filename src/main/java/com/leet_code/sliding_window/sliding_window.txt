Скользящее окно

Используется когда:
- в задаче передается упорядоченная и итерируемая структура данных (массив, строка);
- необходимо найти подпоследовательность в массиве/строке, самое длинное/короткой,
среднее/максимальное/минимальное и т.д..

Метод скользящего окна позволяет улучшить вычислительную сложность до линейной, а по памяти -
до константы.
